# Laptop price predictor
در اینجا من تلاش کرده‌ام اطلاعات تعداد زیادی لپتاپ از فروشگاه های مجازی را استخراج سپس ذخیره کنم و نهایت آن را با الگوریتم ماشین لرنینگ Decision Trees آن هارا تحلیل کنم.

نهایتا مشخصات لپتاپ را ورودی گرفته و قیمت تقریبی آن را ارائه دهم. البته به اندازه کافی این الگوریتم دقیق نیست و برای تحلیل های دقیق قابل استفاده نیست. تنها یک ایده است.

### نحوه استفاده:

در ابتدا از شما پرسیده می‌شود که برای SQL برای ذخیره سازی داده ها استفاده می‌کنید یا خیر. اگر SQL در دستگاه خود ندارید گزینه `n` را بزنید تا اطلاعات مستقیما از وب گرفته شود و بدون ذخیره سازی فرایند ادامه یابد.

اگر `y` را بزنید سپس از شما اطلاعات سرور SQL را میخواهد. نهایتا می‌پرسد که آیا مایل هستید تغییرات را ذخیره کنید و اگر `y` بزنید تصمیماتتان را در فایل `setting` ذخیره می‌کند و دفعه بعد از شما این سوالات را نمی‌کند و بر اساس فایل تنظیمات پیش می‌رود. شما در ابتدای اجرا همچنان می‌تواند دوباره اقدام به تنظیمات کنید.

اگر در سرور خود اطلاعات ذخیره شده باشد از شما می‌پرسد آیا مایل هستید آپدیت کنید داده ها را یا خیر و اگر موافقت کنید دوباره اطلاعات را از وب دریافت کرده و در سرور ذخیره می‌کند.

نهایتا اطلاعات دستگاه مورد نظر را همانند فرمت داده شده وارد می‌کنید و قیمت را در انتها دریافت می‌کنید.
